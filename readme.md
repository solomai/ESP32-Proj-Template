## ESP32 Project Template

Modern C++ project template for ESP-IDF v6.0+ 

Featuring:
- C++ Project Template: Clean structure for modern firmware development.
- Debug-Ready: Integrated OpenOCD configurations for ESP-PROG (JTAG).
- Host Tests: Environment for running unit tests on your PC.
- Device Tests: Template for hardware-in-the-loop testing on the ESP32.

Based on ESP-IDF v6.0.1

Default project standards:
- C standard: 17
- C++ standard: 20
